Factory Game Era 2

After working on the zombie defense typing game, I returned to the factory game with a complete rebuild. The goal was to minimize development time in Unity and maximize time in a proper code editor. To achieve this, I moved most game logic into a separate package with unit tests that could be imported into Unity. This architectural decision proved to be excellent, though I eventually moved on to other projects.
